## About HAL Commons

Library for building and reading JSON HAL resources.


### Maven Dependency

```xml
<dependency>
  <groupId>io.wcm.caravan</groupId>
  <artifactId>io.wcm.caravan.hal.commons</artifactId>
  <version>0.5.0</version>
</dependency>
```
