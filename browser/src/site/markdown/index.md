## About HAL Browser

[HAL Browser][hal-browser] packaged in an OSGi bundle.


### Maven Dependency

```xml
<dependency>
  <groupId>io.wcm.caravan</groupId>
  <artifactId>io.wcm.caravan.hal.browser</artifactId>
  <version>0.5.0</version>
</dependency>
```
